Detailed Description

Interface pour le compteur de vitesse graphique sur TFT ILI9341.

Author
[Ton nom ou initiales]
Date
Mar 28, 2025

Ce module fournit des fonctions pour :

  • dessiner un cadran circulaire avec des graduations,
  • afficher une aiguille animée représentant la vitesse,
  • mettre à jour dynamiquement la vitesse affichée.

Dessine le compteur de vitesse circulaire sur l'écran TFT.

Cette fonction efface l écran, dessine un cercle principal représentant le cadran, puis trace les graduations toutes les 10 unités de vitesse, ainsi que leur valeur numérique.
