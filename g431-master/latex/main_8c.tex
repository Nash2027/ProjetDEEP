Ce programme \+:
\begin{DoxyItemize}
\item Affiche un compteur de vitesse sur écran TFT ILI9341
\item Contrôle un moteur DC en fonction d un potentiomètre
\item Lit un accéléromètre MPU6050 pour détecter un choc
\item Envoie un SMS d alerte via un module SIM808 si un choc est détecté
